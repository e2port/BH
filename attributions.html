Gallery Tools & Tips

This section outlines the tools and techniques used in each gallery, along with some practical tips and insights to help anyone using similar resources.

PDF Documents

  • Tip 1: Google Docs is an amazing way to create professional PDF's for free.
  • Tip 2: If you can keep your Google Drive organized with folders and all projects properly documented, it is a huge time saviour later on.
  • Tip 3: Along with concise description, perhaps add file size next to all files for simple comparison between galleries.

JS-based Apps

  • Tip 1: Implement lazy loading for app thumbnails to boost page performance.
  • Tip 2: All image files should be converted from JPEG/PNG to WEBP; can be done for free with GIMP
  • Tip 3: Can use OpenAI's ChatGPT o30-mini to create, check, and enhance coding.
  • Tip 4: Start with mini-projects to grasp the mechanics and structuring of the coding languages.

Images Gallery

  • Tip 1: GIMP is an amazing free resource to easily modify an image to have a transparent background as well as conversion to webp for superior compression.
  • Tip 2: Blender 3D Open-source Graphics software is amazing, free, and has tons of tutorials on YouTube.
  • Tip 3: Once Blender is mastered, you can create amazing art and use it in conjuction with Unreal Engine to make awe-inspiring movies.

Animations

  • Tip 1: Utilize canvas animations to create immersive visual effects that are relatively light-weight.
  • Tip 2: Optimize performance and CPU usage by controlling the number of active elements and frame rate.
  • Tip 3: Simplistic Fun.

Videos

  • Tip 1: Embed videos using responsive iframes to ensure compatibility across devices.
  • Tip 2: Use YouTube to host videos so you do not take a hit on mp4 storage costs.
  • Tip 3: Microsoft ClipChamp was an amazing free-tool; VSDC is a great open-source video editor that is taking its place for me.
